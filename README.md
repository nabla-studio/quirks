## ğŸ”§ Installation

Node.js is required to run the project.

1. Clone the repository to your local machine:

```bash
git clone https://github.com/nabla-studio/quirks.git
```

2. Init git submodule:

```bash
git submodule init
```

3. Install the dependencies:

```bash
cd quirks
pnpm i
```

4. Build the packages:

```bash
pnpm build
```
